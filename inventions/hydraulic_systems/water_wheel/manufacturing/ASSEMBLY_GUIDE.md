## ğŸ”§ Troubleshooting Guide

### Common Assembly Issues

#### Problem: Wheel Wobble
```
Symptoms: Visible wobble during rotation
Causes: Uneven spoke tension, bent components, poor alignment
Solutions:
1. Check spoke tension with tension meter
2. True wheel using spoke adjustment sequence
3. Replace bent components if necessary
4. Verify bearing alignment and condition

Prevention: Systematic tightening, proper torque application
```

#### Problem: Excessive Vibration
```
Symptoms: High vibration levels, noise, rough operation
Causes: Imbalance, bearing issues, loose components
Solutions:
1. Perform static and dynamic balance checks
2. Inspect bearings for damage or wear
3. Tighten all fasteners to specification
4. Check for damaged or worn components

Prevention: Regular maintenance, proper assembly procedures
```

#### Problem: Low Efficiency
```
Symptoms: Power output below specification
Causes: Poor hydraulic design, excessive friction, leaks
Solutions:
1. Verify water flow rate and head height
2. Check for hydraulic leaks or bypasses
3. Inspect seals and bearings for excessive friction
4. Optimize bucket timing and positioning

Prevention: Proper hydraulic design, quality components
```

#### Problem: Bearing Failure
```
Symptoms: Rough rotation, noise, high temperature
Causes: Contamination, improper installation, overloading
Solutions:
1. Replace bearings with proper specification
2. Clean all mating surfaces thoroughly
3. Apply correct lubrication
4. Verify installation procedure

Prevention: Clean assembly, proper lubrication, correct loads
```
